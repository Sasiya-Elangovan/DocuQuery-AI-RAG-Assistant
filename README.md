# 📄 DocuQuery AI – RAG Assistant

An AI-powered PDF Question Answering Assistant built using LangChain, ChromaDB, Groq LLM, and Streamlit.

## 🚀 Live Demo

https://docuquery-ai-rag-assistant.streamlit.app/

---

## 📌 Features

- Upload PDF documents
- Ask questions from uploaded PDFs
- Semantic search using vector embeddings
- Conversational AI chat
- Fast response generation using Groq LLM
- Streamlit deployment

---

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- ChromaDB
- HuggingFace Embeddings
- Groq API

---

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sasiya-Elangovan/DocuQuery-AI-RAG-Assistant.git
```

### Navigate to Project

```bash
cd DocuQuery-AI-RAG-Assistant
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

#### Windows

```bash
venv\Scripts\activate
```

### Install Requirements

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file and add:

```env
GROQ_API_KEY=your_api_key
```

---

## ▶️ Run Application

```bash
streamlit run app.py
```
